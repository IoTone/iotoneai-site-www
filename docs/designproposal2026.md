## 2. Design Inspiration: Wipeout 2097

### About the Source

Wipeout 2097 (Wipeout XL in North America) was a 1996 futuristic anti-gravity racing game developed by Psygnosis. Its visual identity was created by **The Designers Republic (tDR)**, the legendary Sheffield-based design studio founded by Ian Anderson. tDR's work on Wipeout is widely regarded as one of the most influential examples of graphic design in video game history.

### Key Design Principles from tDR / Wipeout

1. **Brand-as-world-building:** Each racing team (FEISAR, AG Systems, Qirex, Auricom, Piranha, Goteki 45, Assegai, Icaras) was treated as a complete branding exercise with logos, color schemes, and visual language
2. **Typography as identity:** Heavy use of geometric sans-serif fonts (Eurostile, Microgramma, SilverStream) -- futurism encoded in letterforms
3. **High-contrast, neon-on-dark palette:** Deep blacks with electric neons (orange, cyan, magenta, yellow) creating a sense of speed and energy
4. **Minimal ornamentation, maximum impact:** Clean geometric shapes, angular compositions, negative space as a design element
5. **Integration over decoration:** Design is intrinsic to the experience, not applied on top
6. **Corporate-futurism aesthetic:** Fictional megacorps with real-feeling brand systems -- serious enough to be believable, stylized enough to be exciting

### Color Language of Wipeout 2097

The game's visual world uses:
- **Deep black/near-black backgrounds** as the canvas
- **Electric orange** for warnings, speed, energy
- **Cyan/electric blue** for technology, data, navigation
- **Hot magenta/pink** for team accents, highlights
- **Acid yellow-green** for secondary accents
- **Cool white** for typography and clean information display
- **Metallic gray gradients** for structural elements

### Typography Heritage

- **Eurostile Extended** -- the canonical sci-fi typeface, used extensively in menus and UI
- **Microgramma** -- Eurostile's predecessor, tighter and more compressed
- **SilverStream** by Mike Lynch -- used for Wipeout 1 and 2097 logos, lowercase-only
- **Bank Gothic** -- used for supplementary team branding
- The overall approach: geometric, extended, uppercase-dominant, with tight tracking

## 3. Logo Concept A: "VELOCITY MARK"

### Concept Description

A bold, angular wordmark that reimagines "IOTONE" as a speed-driven logotype. The letters are constructed from sharp diagonal cuts and parallelogram shapes, evoking forward motion and anti-gravity racing aesthetics. The "O" letters are replaced with angular hexagonal shapes containing a subtle circuit/signal motif. "JAPAN" appears below in a condensed weight with wide letter-spacing.

### Visual Construction

```
  ╱▀▀▀╲  ╱▀▀▀▀╲ ▀▀▀▀▀▀▀  ╱▀▀▀╲  ▐▌  ▐▌ ▐▀▀▀▀▀▌
 ▐▌   ▐▌ ▐▌    ▐▌   ▐▌   ▐▌   ▐▌ ▐▌▌ ▐▌ ▐▌
 ▐▌   ▐▌ ▐▌    ▐▌   ▐▌   ▐▌   ▐▌ ▐▌▐▌▐▌ ▐▀▀▀▀▌
 ▐▌   ▐▌ ▐▌    ▐▌   ▐▌   ▐▌   ▐▌ ▐▌ ▌▐▌ ▐▌
  ╲▄▄▄╱  ╲▄▄▄▄╱    ▐▌    ╲▄▄▄╱  ▐▌  ▐▌ ▐▄▄▄▄▄▌
               J  A  P  A  N
```

### Key Features

- All letterforms have a 12-degree forward lean (italic/speed angle)
- The "I" in IOTONE doubles as a vertical speed stripe
- The two "O" characters use a hexagonal frame with an inner node pattern (resembling a simplified circuit/network)
- "JAPAN" is set in ultra-wide tracking (400%) as a grounding anchor
- A single horizontal "speed line" extends from the left edge through the entire mark
- The overall shape fits within a 5:1 horizontal rectangle

### Color Application

- Primary: Electric Orange `#FF6B00` wordmark on black
- Alternate: White wordmark on black with orange speed line accent
- Monochrome: All white on black or all black on white

### Personality

Speed. Precision. Forward momentum. This mark says "we move fast and build the future."

## 4. Logo Concept B: "SIGNAL GRID"

### Concept Description

A modular, grid-based logo system inspired by the team logo approach of The Designers Republic. The mark consists of two elements: (1) a square icon containing an abstract "IO" monogram built from geometric grid elements -- circles, lines, and nodes that suggest both IoT connectivity and digital signal processing, and (2) a clean wordmark "IOTONE JAPAN" set in extended geometric sans-serif.

### Visual Construction

```
 ┌─────────────────┐
 │  ●───────●       │
 │  │       │       │     I O T O N E
 │  │   ●───┤       │     ─────────────
 │  │   │   │       │     J   A   P   A   N
 │  ●───●   ●       │
 │          │       │
 │      ●───●       │
 └─────────────────┘
```

### Key Features

- The square icon is built on a 6x6 grid
- The "I" is represented by a vertical line of 3 connected nodes
- The "O" is represented by a square circuit path of 4 connected nodes
- Nodes are solid circles (dots), connections are straight lines at 0/90-degree angles
- The grid system allows the icon to scale from favicon (16px) to billboard
- The wordmark uses Eurostile Extended or similar geometric extended sans-serif
- A thin ruled line separates "IOTONE" from "JAPAN"

### Color Application

- Primary: Cyan `#00F0FF` icon + White wordmark on deep black `#0A0A0F`
- Alternate: Magenta `#FF2D78` icon for secondary/event use
- The icon can rotate through accent colors for different divisions or contexts
- Monochrome: Solid white or solid black

### Personality

Connected. Systematic. Technical precision with visual clarity. This mark says "we are the infrastructure of the future."

## 5. Logo Concept C: "NEURO CIRCUIT"

### Concept Description

A dynamic, asymmetric logo mark that fuses Japanese typographic sensibility with Wipeout-era futurism. The mark is a stylized kanji-inspired abstract glyph that suggests both "one" (the "one" in IoTone) and a neural/circuit pathway. It is paired with a stacked bilingual wordmark: "IOTONE" in Latin characters and a Japanese katakana rendering below.

### Visual Construction

```
        ▄▄████▄▄
      ▄█▀      ▀█▄          I O T O N E
     █▀   ╱▔▔╲   ▀█         ──────────────
    █▀   ╱    ╲   ▀█        アイオートーン ジャパン
   █▀   ╱  ●●  ╲   ▀█
  █▀   ╱  ●  ●  ╲   ▀█
  █   ╱__●____●__╲   █
  █▄▄▄▄▄▄▄▄▄▄▄▄▄▄▄▄▄█
```

### Key Features

- The glyph mark is an abstract form that reads differently at different scales: at small sizes it's a bold geometric shape; at large sizes the internal circuit/neural pathways become visible
- The internal pattern consists of branching lines with node terminals (dots), suggesting both a circuit board trace and a neural network
- The outer envelope is an asymmetric shield/blade shape -- one side curved (organic/human), one side angular (technology/machine)
- The bilingual wordmark stack places English and Japanese in visual parity
- "IOTONE" in Eurostile Extended Bold, "アイオートーン ジャパン" in a matching geometric Japanese typeface (e.g., A-OTF Gothic MB101 or Noto Sans JP Black)

### Color Application

- Primary: Gradient from Electric Orange `#FF6B00` to Hot Magenta `#FF2D78` on the glyph mark, white wordmark on black
- Alternate: Solid Acid Yellow `#D4FF00` mark for high-impact applications
- Digital: The gradient can be animated (slow color shift) for web/app use
- Monochrome: Solid white on black

### Personality

Duality. East meets West. Human intelligence amplified by technology. This mark says "we bridge worlds and connect minds."

## 7. Typography System

### Primary Typeface: Eurostile Extended

The canonical futuristic typeface, used extensively in Wipeout and throughout sci-fi design since the 1960s. Its squared proportions and extended width create a sense of technological authority.

| Weight | Use |
|--------|-----|
| **Eurostile Extended Bold** | Headlines, logo wordmark, hero text |
| **Eurostile Extended Medium** | Subheadings, navigation |
| **Eurostile Extended Regular** | Body emphasis, labels |

**Web alternatives (free):** Orbitron, Exo 2, Rajdhani, Michroma (Google Fonts)

### Secondary Typeface: Inter or DM Sans

A clean, modern geometric sans-serif for body text readability.

| Weight | Use |
|--------|-----|
| **Inter Medium** | Body text (English) |
| **Inter Regular** | Long-form content |
| **Inter Bold** | Inline emphasis |

### Japanese Typeface: Noto Sans JP or M PLUS 1p

Geometric Japanese typeface that pairs well with Eurostile's squared character.

| Weight | Use |
|--------|-----|
| **Noto Sans JP Bold** | Japanese headlines |
| **Noto Sans JP Medium** | Japanese navigation |
| **Noto Sans JP Regular** | Japanese body text |

### Type Scale

| Level | Size | Weight | Tracking | Use |
|-------|------|--------|----------|-----|
| Display | 72px / 4.5rem | Eurostile Extended Bold | +0.05em | Hero headlines |
| H1 | 48px / 3rem | Eurostile Extended Bold | +0.03em | Page titles |
| H2 | 32px / 2rem | Eurostile Extended Medium | +0.02em | Section headers |
| H3 | 24px / 1.5rem | Eurostile Extended Medium | +0.02em | Subsection headers |
| Body | 16px / 1rem | Inter Regular | 0 | Paragraph text |
| Caption | 14px / 0.875rem | Inter Regular | +0.01em | Labels, metadata |
| Micro | 11px / 0.6875rem | Eurostile Extended Medium | +0.15em | Badges, tags, all-caps labels |

### Typography Rules

1. Headlines are always **UPPERCASE** in Eurostile Extended
2. Body text is **sentence case** in Inter/Noto Sans JP
3. Minimum letter-spacing on Eurostile: +0.02em (it needs room to breathe)
4. Line height for body text: 1.6; for headlines: 1.1
5. Maximum body text width: 72ch
6. Japanese body text line height: 1.8 (standard for Japanese readability)

## 8. Icon System

### Icon Design Principles

Following the Wipeout/tDR approach, icons should feel like they belong to a fictional megacorp's internal design system:

1. **Geometric construction:** All icons built on a 24x24 grid using circles, rectangles, and 45-degree diagonals only
2. **Consistent stroke weight:** 2px stroke at default 24px size
3. **Node terminals:** Line endpoints use small circular terminals (dots) -- the signature "circuit node" motif
4. **No rounded corners** on rectangular elements (sharp = fast)
5. **Monoline style** -- no fills, outlines only (fills used only for active/selected states)

### Core Icon Set

| Icon | Name | Use |
|------|------|-----|
| Circuit node cluster | **IoT / Connected** | IoT services, connectivity features |
| Eye with scan lines | **Vision / AR** | AR/VR services |
| Shield with circuit trace | **Security** | Home security, data protection |
| Pulse wave + cross | **Health / Medical** | Preventive medical tech |
| Cube wireframe | **VR / Spatial** | Virtual reality, spatial computing |
| Lightning bolt (angular) | **AI / Processing** | AI services, computation |
| Globe with grid overlay | **Global / Network** | International services, STEM education |
| Hexagonal badge | **Certification** | Partner certifications, quality marks |

### Favicon & App Icon

- **Favicon (16x16, 32x32):** Simplified "IO" monogram from Signal Grid concept -- two connected nodes
- **App Icon (512x512):** Full icon mark on Void Black background with subtle Circuit gradient
- **Touch Icon:** Icon mark in Neon Orange on Void Black, with rounded corners per platform spec

### Print Icons

- For print at small sizes (< 12mm), use solid filled versions instead of stroked
- Minimum clear space around any icon: 50% of icon width on all sides

### Responsive Logo Behavior

| Viewport | Logo Display |
|----------|-------------|
| >= 1024px | Full icon mark + "IOTONE JAPAN" wordmark |
| 768-1023px | Full icon mark + "IOTONE" wordmark (drop "JAPAN") |

### Animation Guidelines

- Subtle grid/line animations on hover (CSS only, no JS required)

- Color transitions: 0.2s ease-in-out
- No heavy particle effects or WebGL -- keep it fast and clean, like the game itself
- Preferred motion: horizontal translations (speed/motion) over vertical or rotational

## Recommendation

**Logo Concept B: "SIGNAL GRID"** is recommended as the primary direction. It offers:

- The strongest scalability from favicon to billboard
- The most direct connection to IoT/connectivity as a visual metaphor
- The closest alignment with The Designers Republic's modular team branding approach
- The cleanest bilingual application (the icon mark is language-agnostic)
- The most distinctive silhouette at small sizes

Concepts A and C should be explored as secondary marks or campaign-specific variations.
